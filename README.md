# electron-bug-demo

** IME focus bug on macOS mojave**


## How to Repro

0. upgrade to macOS mojave
1. npm install
2. npm start
3. app is running. click another app to blur electron app
4. click Tray icon to focus app
5. type something with IME ( third party or not )
